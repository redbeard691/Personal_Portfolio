Mimics

Date: April, 2025

Languages: Javascript

Frameworks: Node.js, Express.js

Tools: Visual Code, Git

This project was developed during a 24-hour WSU hackathon focused on the theme of internet privacy and safety. Challenging myself to work solo, I created an AI that tracks and mimics a user's browsing behavior to mask their true online activity. By the end of the event, I had a working proof of concept that was able to have multiple copies of me and could go to different pages on the same website I was on. This project pushed my skills and highlighted areas for future development and if I had more time would be able to flush out more features and make it a more complete project.

Animal Rescue Website

Date: May, 2025

Languages: HTML, CSS, Javascript

Frameworks: Node.js, Express.js

Tools: Git, Visual Code

Developed a community-focused animal rescue website with Node.js and Express.js, enabling users to post notices for lost and found pets. The platform allows for user profile creation, which includes a secure messaging system for direct communication between individuals. In addition to the core pet-finding features, the site also provides articles on animal care and has an administrative structure with distinct privileges for admins, shelters, and regular users, allowing for content management and moderation.
