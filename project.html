Pirate Mayhem

Date: April, 2025

Languages: C#

Frameworks: .Net, Monogame

Tools: Git, Visual Studio

Repository: N/A

A 2D arcade-style game developed with the Monogame framework for a university-level game for a Software Design and Architecture class. As the captain of a lone pirate ship, players must strategically navigate chaotic battles against swarms of goblin-crewed ships, mythical krakens that hurl massive boulders, and enigmatic spaceships attempting to steal your gold.

The core design challenge was to maintain performance under increasingly chaotic conditions. We focused on implementing design patterns to optimize efficiency, particularly as the number of on-screen objects, such as cannonballs and enemy projectiles, escalated. We specifically utilized patterns like singleton, Builder pattern and Command Patterns to manage bullet creation and destruction, significantly reducing the performance overhead of spawning new objects in real-time. This emphasis on optimized game architecture ensured a smooth and scalable gameplay experience.

Spread Sheet Application

Date: December 2023

Languages: C#

Frameworks: .Net, Windform

Tools: Nunit, Git

Repository: Spread Sheet Applicaion

This project is a simple yet robust spreadsheet application designed to offer essential data management and calculation functionalities. The primary focus of the application is to provide a seamless user experience for basic data entry, while ensuring the integrity of all calculations through a dynamic update system.

The application allows users to create, load, and save spreadsheet files, enabling them to easily manage their work. A key feature is the ability to highlight individual cells for organizational purposes. Most critically, the system is engineered to automatically and instantly re-calculate and update any cell affected by a change, ensuring all arithmetic operations are consistently and accurately reflected across the entire sheet. This dynamic updating process provides real-time feedback and maintains data consistency, making it a reliable tool for simple data analysis.
